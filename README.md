# HP-preview

KiT Works が近隣の個人店・小規模店舗へ営業連絡を行う際に、スマホで見やすい1ページ店舗案内ページの完成イメージを伝えるためのHTMLデモです。

架空店舗「ひよりカフェ」を題材に、実店舗の公式案内ページとしてそのまま流用しやすい情報設計、CTA、メニュー、アクセス、問い合わせ導線をまとめています。

## このHTMLデモの用途

- Instagram DMやメールでURLを共有し、制作イメージを一目で伝える
- 「今の発信が悪い」と指摘するのではなく、公式案内ページの使い道を自然に見せる
- 個人店向けに、低負担で公開できる1ページサイトの雰囲気を示す
- GitHub Pagesなどで公開し、スマホから確認できる営業サンプルとして使う

## 主な内容

- ファーストビュー、予約・問い合わせCTA、Instagram CTA
- 店舗紹介、選ばれる理由、メニュー
- 初めての方向けの案内、営業時間、アクセス
- 電話、Instagram、メール、フォーム風UI
- OGP、Twitter Card、favicon、JSON-LD構造化データ
- スマホ下部の固定CTA

## ローカルでの確認方法

`index.html` をブラウザで開くと確認できます。

```text
HP-preview/index.html
```

静的なHTML/CSS/JavaScriptのみで構成しているため、基本的にはローカルサーバーは不要です。

## GitHub Pagesで公開する方法

1. GitHubにpushする
2. リポジトリの `Settings` を開く
3. `Pages` を選択する
4. Sourceを `Deploy from a branch` にする
5. Branchを `main`、フォルダを `/root` に設定する
6. 発行されたURLを営業DMやメールに記載する

公開URLを変更する場合は、`index.html` 内の `canonical`、OGP、Twitter Card、JSON-LDのURLも合わせて差し替えてください。

## 実店舗向けに差し替える箇所

- 店舗名、キャッチコピー、紹介文
- 電話番号、メールアドレス、Instagram URL
- メニュー名、説明、価格
- 営業時間、定休日、ラストオーダー
- 住所、Google Mapリンク、駐車場、最寄り駅や目印
- フォームの送信先や予約導線
- OGPタイトル、説明文、画像URL
- JSON-LDの店舗情報
- 色、余白、文章のトーン

## 差し替え時の注意点

- 電話番号やメールは必ず実店舗の公開可能な連絡先にする
- Google Mapリンクは住所検索ではなく、店舗の正式なGoogleマップURLに差し替えると安定する
- メニュー価格は税込・税抜、店内・テイクアウトの違いを明記する
- 定休日や臨時休業の案内方法を決めておく
- OGP画像を変更した場合は、SNSのキャッシュ更新も確認する
- フォームを実送信にする場合は、迷惑メール対策と個人情報の扱いを確認する
