# HP-preview

近隣の個人店・小規模店舗に向けて、スマホで見やすい1ページ店舗案内ページの制作イメージを伝えるためのHTMLデモです。Instagram DMやメールでURLを共有し、「このくらいの小さな公式案内ページなら使えそう」と自然に感じてもらうことを目的にしています。

## HTMLデモの概要

- 架空店舗「ひよりカフェ」の店舗案内ページ
- スマホ表示を主軸にした1ページ完結の構成
- ファーストビュー、店舗紹介、メニュー、営業時間、アクセス、お問い合わせを掲載
- 外部画像を使わず、CSSの図形・色面・背景表現で写真風の雰囲気を作成
- 実在店舗の写真・ロゴ・文章に依存しない営業用サンプル

## ローカルでの確認方法

`index.html` をブラウザで開くと確認できます。

```text
HP-preview/index.html
```

静的なHTML/CSS/JavaScriptのみで構成しているため、ローカルサーバーを起動しなくても表示できます。

## ファイル構成

```text
HP-preview/
├── index.html
├── styles.css
├── script.js
└── README.md
```

## 実店舗向けに流用する場合の差し替え箇所

- 店舗名、キャッチコピー、紹介文
- 電話番号、Instagram URL、メールやフォームの送信先
- メニュー名、説明文、価格
- 営業時間、定休日
- 所在地、Google Mapsのリンク
- 店舗の雰囲気に合わせた色、余白、文章のトーン

## GitHub Pagesで公開する場合

GitHubにpushしたあと、リポジトリの `Settings` から `Pages` を開き、公開元を `main` ブランチのルートに設定すると公開できます。公開後に発行されるURLをDMやメールに記載すれば、スマホから直接デモを確認できます。
